O que são tags HTML?

Tags HTML são instruções que informam ao navegador como formatar o texto. Aqui estão alguns exemplos práticos:

Exemplos de Tags HTML

  • Tag de Título: 
    <h1>Título Principal</h1>
  • Tag de Parágrafo: 
    <p>Este é um parágrafo.</p>
  • Tag de Imagem: 
    <img src="imagem.jpg" alt="Descrição da imagem">
  • Tag de Link: 
    <a href="https://www.exemplo.com">Clique aqui</a>
  • Tag de Lista: 
    <ul><li>Item 1</li><li>Item 2</li></ul>
